# web-api
Public API documentation for Brave New Coin.

## Install ReDoc CLI
```
npm i -g redoc-cli
```

## Build documentation
```
redoc-cli bundle openapi.yaml --output build/index.html --cdn --title "BraveNewCoin API documentation"
```
